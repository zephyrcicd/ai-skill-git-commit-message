# Tell AI

Use this file when you want another AI agent to install or update this repository for you.

## Install Instructions

Tell the AI:

```text
Install the Git Commit Message Skills repo from https://github.com/zephyrcicd/ai-skill-git-commit-message for the platform I am currently using. Follow the platform-specific install guide from this repository:
- Claude Code: docs/install-claude-code.md
- Codex: docs/install-codex.md
- OpenCode: docs/install-opencode.md
- Gemini CLI: docs/install-gemini-cli.md

If the platform is Codex, Claude Code, or OpenCode, prefer the documented git-based install flow from this repo. If the platform is Gemini CLI, update ~/.gemini/GEMINI.md so Gemini knows where to load the skill files from.

After installation, verify the installed skill names are available:
- ggm
- ggm-p
```

## Update Instructions

Tell the AI:

```text
Update my existing installation of https://github.com/zephyrcicd/ai-skill-git-commit-message by following docs/update.md for the platform I am using. After updating, verify that ggm and ggm-p are still available.
```
